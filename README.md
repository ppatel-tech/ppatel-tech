# Featured Projects

###  [FinSight AI — Intelligent Financial Backend System](https://github.com/ppatel-tech/finsight-ai.git)
A production-grade financial management backend designed to secure transaction data, monitor budgets in real time, and deliver context-aware financial insights.
*   **Core Stack:** Java 21, Spring Boot 3, Spring Data JPA, PostgreSQL, Redis, Docker, LangChain4j, Flyway, Render.
*   **Key Engineering Implementation:** Built JWT-based authentication with refresh token rotation, integrated an event-driven budget monitoring subsystem (Gmail SMTP alerts), optimized queries using Redis caching, and managed schema migrations via Flyway.
